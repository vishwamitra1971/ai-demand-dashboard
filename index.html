A signal: EU high-risk AI rules (potential 16-month delay via Digital Omnibus) — if energy consumption restrictions on DCs emerge.
C signal: US deregulatory stance removes federal constraints. Geopolitical AI arms race (US/China) prevents meaningful restraint.

Scenario-Contingent Actions
If Scenario A Emerges
2+ triggers firing simultaneously. Currently: Edge AI (T4) and GPU lease rates approaching A threshold (M2).
IMPACT
→ Long-term DC contracts face early termination pressure
→ Overcapacity in ERCOT and PJM
→ Pricing pressure on merchant generation
ACTION
→ Halt uncommitted Phase 3+ builds
→ Renegotiate contracts — add efficiency adjustment clauses
→ Pivot fleet value to grid reliability & capacity payments
→ Accelerate consumer/VPP revenue diversification

Scenario B — Execute Plan
Baseline. Look for any 3+ indicators deviating from B confirmation ranges.
IMPACT
→ Contracted DC pipeline executes on schedule
→ Steady predictable demand growth supports fleet utilization
→ Market share, not market size, determines winners
ACTION
→ Execute phased buildout per plan (GE Vernova/Kiewit JV)
→ Maintain 12–15% unlevered IRR discipline on all new contracts
→ Build 5–10 year contract duration book (not 20–25)
→ Continue BYOP positioning for hyperscaler relationships
If Scenario C Emerges
3+ triggers firing. Currently: Capex (M1), interconnect queues (M3), vacancy (M4), GPU shipments (A1) already firing.
IMPACT
→ Scarcity pricing viable ($100+/MWh contracts)
→ Build-out cannot keep pace; BTM solutions proliferate
→ Regulatory/political backlash if DC exceeds 12% of US electricity
ACTION
→ Accelerate all available capacity under contract
→ Increase pricing 15–20% on new contracts
→ Expand BYOP aggressively — BTM is the only path to speed
→ Pre-position sites, interconnection rights, gas supply for rapid deployment
